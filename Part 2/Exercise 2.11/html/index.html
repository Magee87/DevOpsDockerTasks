The Force is Strong with Docker

Welcome to Star Wars-style simple website! Here, we'll explain how we used Docker to create a development environment for our project.

Step 1: Dockerfile

First, we created a Dockerfile to specify the configuration of our container. We chose to use an Alpine-based image and Node.js installed. We also specified the working directory and copied our project files into the container.

Step 2: docker-compose.yml

Next, we created a docker-compose.yml file to define our services. We specified the name of our service, the build context (where our Dockerfile is located), and the ports to expose.

Step 3: Start the container

Finally, we started the container using the docker-compose up command. This created our development environment accessible on our local machine at localhost:3000

Step 4: Site data

Meanwhile, this site data is collected from html directory. First we fetch index.html and then we collect all other files like this style.css file what gives little bit style to this site.
